# Welcome to my dotfiles

*perfection* is not something you will find here

## Steps to install
```bash
wget -qO- https://raw.githubusercontent.com/CameronHonis/dotfiles/main/.setup/bootstrap.sh | bash
```
or using curl:
```bash
curl -sSL https://raw.githubusercontent.com/CameronHonis/dotfiles/main/.setup/boostrap.sh | bash
```
